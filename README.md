# 常微分方程期末复习手册 (ODE Review Guide)

这是一个基于纯前端（HTML + CSS + JS）打造的**常微分方程（Ordinary Differential Equations）期末复习专属网页**。
本项目旨在将手写的纸质笔记数字化、结构化，提供一个支持数学公式精准渲染、排版美观、支持暗黑模式的复习环境，帮助在考前进行最高效的知识梳理。

## ✨ 核心特性

- **现代 UI 设计**：采用全局暗色系（Dark Mode）与卡片式毛玻璃（Glassmorphism）效果，视觉舒适，降低复习时的疲惫感。
- **完美的公式渲染**：底层接入强壮的 `MathJax 3.0` 引擎，精准解析和渲染所有复杂的微积分符号与线性代数矩阵。
- **丝滑的侧边导航**：内置平滑滚动（Smooth Scroll）与基于 `IntersectionObserver` 的目录联动高亮，方便在不同知识模块间快速跳转。
- **纯静态免安装**：无需配置任何后端或 Node 环境，直接双击 `index.html` 即可在任意浏览器中完美运行。

## 📚 涵盖知识点

本复习手册目前收录了以下核心考点：

1. **基本概念介绍**（可分离变量、齐次方程、积分因子法等）
2. **一阶方程解法**（一阶线性微分方程的标准解法步骤）
3. **解的存在唯一性定理**（利普希茨条件的严谨判定：闭区间连续或开区间有界）
4. **常用麦克劳林级数**（$e^x, \sin x, \cos x, \ln(1+x), \frac{1}{1-x}$ 的级数展开与收敛域）
5. **线性微分方程组**：
   - 常系数齐次方程组的**特征值法**（求特征根、特征向量）
   - 非齐次方程组的**常数变易法**（求逆矩阵、矩阵乘法与逐元积分详细拆解）
6. **易错点与避坑记录**（深入剖析积分常量处理、移项符号遗漏等高频手误，附带代入检验法）

## 🚀 如何使用

1. **本地使用**：克隆或下载本仓库，找到根目录下的 `index.html`，双击用浏览器（推荐 Chrome / Edge）打开即可。
2. **在线部署**：可以直接将本文件夹拖拽至 [Netlify Drop](https://app.netlify.com/drop) 实现一键免费云端部署，生成可供手机查看和分享的永久在线链接。

## 🤝 贡献与修改

如果你有更多想要补充的笔记、易错点或例题：
